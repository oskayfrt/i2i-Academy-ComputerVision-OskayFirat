Computer Vision - Parmak Sayma Uygulaması
Bu projede Python, OpenCV ve MediaPipe kullanılarak gerçek zamanlı bir parmak sayma uygulaması geliştirilmiştir. Kamera görüntüsündeki el algılanır, 21 el eklem noktası belirlenir ve açık parmakların sayısı ekranda gösterilir.
Kullanılan Teknolojiler
Python
OpenCV
MediaPipe Hand Landmarker
Kurulum
pip install opencv-python mediapipe
Çalıştırma
python main.py
Programdan çıkmak için Q tuşuna basılabilir.
Proje Dosyaları
main.py: Uygulamanın kaynak kodu
hand_landmarker.task: MediaPipe el algılama modeli
